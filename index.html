Điều khiển

Thí sinh chỉ cần mở link là có thể bắt đầu thi.
Đang tải gói câu hỏi từ questions.xlsx...
Gói câu hỏi được lấy từ file questions.xlsx đặt cùng thư mục với file HTML. Mỗi lượt thi chọn ngẫu nhiên 20 câu từ ngân hàng đề.
Đổi mật khẩu tại biến ADMIN_PASSWORD trong mã nguồn HTML.
Định dạng Excel chuẩn: các cột ID | Question | A | B | C | D | Correct | Explanation. Cột Correct nhập một trong 4 giá trị: A, B, C, D.

Sẵn sàng tổ chức thi

Hệ thống đang chờ lựa chọn chế độ làm việc.
Khi đưa bộ file này lên hosting (ví dụ GitHub Pages), thí sinh chỉ cần mở link là làm bài được.
Để Admin xem kết quả tổng hợp, cần dán URL của Google Apps Script Web App vào biến SCRIPT_URL trong file HTML. Phần mã Apps Script sẽ được cung cấp riêng.
